About me

I am a french Web and 3D software programmer.

My studies and personal projects are centered around the development of game engines, 3D graphics software and programming language theory.

My professional experience is about the development of website backends and internal web tools.


Through my two-year university diploma and my professional bachelor degree, I learnt about the development of 3D applications, the game engines behind those applications (Unreal Engine, Unity, Godot...), the low-level graphics components of those game engines (OpenGL, Vulkan...) and the graphics pipeline used by those low-level libraries.


I worked at several web agencies as a backend web developer, with PHP and the Symfony web framework. Occasionally, my job was to develop internal web tools to help developers in their tasks, or to automate some processes.


As said above, i am also studying the theory behind programming languages. Programming languages are the first and main tools used by programmers, and some languages make some problems easier to resolve.

While PHP and Javascript are still kings in the web development world, some programming languages with various qualities grew in popularity. Erlang/Elixir is a fault-tolerant and concurrent by default programming language, making it a really attractive for client/server software. Facebook developed the ReasonML language on top of the Ocaml language, which offers an expressive syntax and a static type-checker to the Javascript world.

Some of those new languages make the development of applications better and safer: less bugs, a better architecture, better development tools...

Sometimes, being able to develop a small domain-specific language to solve a particular problem can be a good idea, that is why I am studying this field.
